Tentang Aplikasi

Dirancang untuk kebutuhan akademik yang cepat, rapi, dan mudah dipakai.

Aplikasi berjalan sepenuhnya di browser dengan HTML5, CSS3, JavaScript murni, Canvas API, dan mock API JSON lokal sehingga mudah dipindahkan ke REST API sungguhan.

  • Tanpa framework, tanpa backend wajib.
  • Editor drag, zoom, rotate, dan reset.
  • Export PNG transparan resolusi tinggi.
  • Responsif untuk desktop, tablet, dan mobile.
