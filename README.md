# Twibbon-FTI-UNISKA
Aplikasi twibbon untuk ucapan pada fakultas teknologi informasi universitas islam kalimantan syekh Muhammad Arsyad al Banjari Banjarmasin 

Cara penggunaan:
1. Memilih template twibbon.
2. Upload foto.
3. Mengatur posisi foto (drag, zoom, rotate).
4. Menghasilkan gambar twibbon.
5. Download hasil dalam format PNG kualitas tinggi.
